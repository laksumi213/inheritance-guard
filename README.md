# 相続実務・AI送信前 匿名化ガバナンスツール (v1.3)

相続実務（遺言作成、遺産分割協議、戸籍読み取り等）において、ChatGPT等の外部AIを安全かつ効率的に活用するための「実務家のための防御ツール」です。

## 🛠 本ツールの目的
外部AIを利用する際の最大の懸念である「個人情報の漏洩リスク」を、物理的に遮断します。
送信前に「氏名・続柄・住所（地番対応）・口座番号・日付」を記号（トークン）に一括置換し、AIからの回答後に元の実名に復元します。

## 🛡 安全性と透明性へのこだわり
本ツールは「実務家が自ら責任を持ってAIを使う」ための証拠（エビデンス）を残せるよう設計されています。

1. **ローカル完結型**: 入力されたデータはインターネットに送信されません。あなたのPC内だけで処理されます。
2. **クリーンビルドの証明**: 開発者のPC環境に依存せず、GitHub Actions（公式サーバー）上で公開ソースコードから直接ビルドされています。
   - [最新のビルド実行ログと成果物の証明](https://github.com/laksumi213/inheritance-guard/actions)
3. **ウイルススキャン済み**: 世界中のエンジンによる安全診断を実施済みです。
   - [VirusTotal 診断結果](https://www.virustotal.com/gui/file/41831a85c858b7ad6d430efa1eb6df98ab2f894dda11a7b7cc176a8cc5092ec2/detection)
   ※一部のソフトで誤検知（False Positive）が出る場合がありますが、GitHub Actionsによる生成ログが安全性の証明となります。

## 🚀 使い方（Windows用）

1. **ダウンロード**: 右側の「Releases」から最新版の `相続AI匿名化ツール.exe` をダウンロードしてください。
2. **実行**: ダウンロードしたファイルをダブルクリックして起動します。
3. **検知・編集**: 左側に原稿を貼り付け「一括検知」を実行。リストに抽出された項目を確認し、必要に応じて**ダブルクリックで直接修正**したり、不要な行を削除します。
4. **匿名化**: 「匿名化 ➔ コピー」を押すと、クリップボードに匿名化されたテキストがコピーされます。
5. **AI相談**: AIからの回答を右側に貼り付け「実名に復元」を押せば、実名入りの回答に即座に復元されます。

## 📝 主な機能（v1.3 アップデート）
- **実務特化型・高度な自動検知**: 
    - 氏名だけでなく「長女」「遺言執行者」などの**続柄・肩書き**を考慮して抽出。
    - 不動産登記特有の**「地番」表記**や複雑な空白を含む住所を検知。
- **実務ノイズ除去ロジック**: 
    - 「遺言者の〜」「〜の持分」といった、名前ではない定型句を自動で判別し、検知対象から除外。
- **リストの直感編集**: 
    - 抽出されたリストの「実名」や「トークン」を**ダブルクリックで即座に手書き修正**可能。
- **ガバナンスログ**: `anonymization_log.txt` を自動生成し、匿名化の証跡を保存。
- **設定保存**: 案件ごとの置換リストをJSON形式で保存・読み込み可能。
