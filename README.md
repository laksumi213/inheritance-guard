# 相続実務・AI送信前 匿名化ガバナンスツール

相続実務（遺言作成、遺産分割協議など）において、ChatGPT等の外部AIを安全に活用するための「実務家のための防御ツール」です。

## 🛠 本ツールの目的
外部AIを利用する際の最大の懸念である「個人情報の漏洩リスク」を、物理的に遮断します。
送信前に「実名・住所・口座番号・日付」を記号（トークン）に一括置換し、AIからの回答後に元の実名に復元します。

## 🛡 安全性と透明性へのこだわり
本ツールは「実務家が責任を持ってAIを使う」ための証拠（エビデンス）を残せるよう設計されています。

1. **ローカル完結型**: 入力されたデータはインターネットに送信されません。あなたのPC内だけで処理されます。
2. **クリーンビルドの証明**: 開発者のPC環境に依存せず、GitHubの公式サーバー（GitHub Actions）上で公開ソースコードから直接ビルドされています。
   - [ビルド実行ログと成果物の証明](https://github.com/laksumi213/inheritance-guard/actions/runs/24353022380)
3. **ウイルススキャン済み**: 世界中のエンジンによる安全診断を実施済みです。
   - [VirusTotal 診断結果](https://www.virustotal.com/gui/file/41831a85c858b7ad6d430efa1eb6df98ab2f894dda11a7b7cc176a8cc5092ec2/detection)
   ※一部のソフトで誤検知（False Positive）が出る場合がありますが、GitHub Actionsによる生成ログが安全性の証明となります。

## 🚀 使い方（Windows用）

1. **ダウンロード**: 右側の「Releases」または[こちら（Artifacts）](https://github.com/laksumi213/inheritance-guard/actions/runs/24353022380)から `Windows-EXE-Package` をダウンロードしてください。
2. **実行**: 中にある `相続AI匿名化ツール.exe` をダブルクリックして起動します。
3. **匿名化**: 左側に原稿を貼り付け「一括検知」→「匿名化 ➔ コピー」を押します。
4. **AI相談**: クリップボードにコピーされた匿名テキストをAIに貼り付けます。
5. **復元**: AIの回答を右側に貼り付け「実名に復元」を押せば、実務書類として完成します。

## 📝 機能一覧
- **高度な自動検知**: 氏名、住所（ゆらぎ対応）、日付、口座番号を正規表現で抽出。
- **ガバナンスログ**: `anonymization_log.txt` を自動生成し、匿名化の証跡を保存。
- **残存チェック**: 匿名化漏れ（「市」や「様」など）を検知して警告。
- **設定保存**: 案件ごとの置換リストをJSON形式で保存・読み込み可能。
